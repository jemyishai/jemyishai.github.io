    Blanche Digital Library

    I was the lead developer on the Blanche Castleman Link Digital Library. The project was part of a digital libraries course that digitized a scrapbook from an early 19th century Simmons Student using Omeka, an open source digital library tool. Leading the web development team meant creating a custom 'look and feel' for the library, creating a workflow for other students, and the ingestion of meticulous metadata.

    • Client: Blanche Library
    • Date: Fall 2015
    • Service: Web Development and Web Design for Notable Women of Simmons Project
